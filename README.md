# Protein Haven

A front-end website for Protein Haven, a local protein shake and nutrition bar. Built with plain HTML and CSS.

## Pages

- **Home** - Hero section with intro, stats, and call to action
- **About** - Story and values behind Protein Haven
- **Contact** - Contact details and inquiry form
- **Hours** - Opening hours

## Tech Stack

- HTML5
- CSS3
- No frameworks or libraries

## Project Structure

```
ProteinHaven/
├── index.html
├── about.html
├── contact.html
├── hours.html
└── assets/
    └── images/
```

## How to Run

Just open `index.html` in your browser. No installs needed.
